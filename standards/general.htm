iterami/Docs.htm/Standards/General
  • 24 hour clock must always be used.
  • Numbers should use specific suffixes:
    Thousand 123k
    Million 654m
    Billion 789b
    Trillion 210t
    Quadrillion 345q
  • Timezone default is UTC.
  • Weeks begin on monday.
  • Words are replaced with more compact versions whenever possible:
    Original
    Word     		Replacement Notes
    arguments args Values passed to a function.
    combination combo
    configuration config
    conversation convo
    corporation corp
    difference diff
    documentation docs
    first, second, third, ... 1st, 2nd, 3rd, ...
    HTML HTM
    introduction intro
    repository repo
    statistics stats
    though tho
    through thru
