iterami/Docs.htm/Standards/HTM
  • All pages must include:
    • <!doctype html>
    • <meta charset=utf-8>
    • <title>A Title</title>
    • <link href=a/css/file.css rel=stylesheet>
  • External links:
    • Must include the "external" CSS class.
    • Must include "rel=noopener".
  • Mobile friendly pages must include:
    • <meta content="width=device-width,initial-scale=1" name=viewport>
