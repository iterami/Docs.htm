iterami/Docs.htm/Repositories/Browser.gtk
  • Installation:
    1. "git clone https://github.com/iterami/Browser.gtk.git --depth 1"
    2. "sudo aptdcon -i libgtk-3-dev"
    3. "git clone https://github.com/iterami/common.git --depth 1" into the same folder as "Browser.gtk".
    4. Use your terminal to navigate to the "Browser.gtk" folder and "make" it.
    5. Run the compiled "~/.iterami/browser".
  • Updating:
    1. Close "Browser.gtk".
    2. "git pull" the "common" and "Browser.gtk" repositories that you cloned.
    3. Use your terminal to navigate to the "Browser.gtk" folder and "make" it.
    4. Run the compiled "~/.iterami/browser".
  • Uninstallation:
    1. Use your terminal to navigate to the "Browser.gtk" folder.
    2. "make clean"
    3. Delete the cloned "Browser.gtk" folder.
    4. Optionally remove "common" and optionally "sudo aptdcon -r libgtk-3-dev".
    5. If you have no other iterami GTK repositories installed, you can delete the "~/.iterami/" folder.
