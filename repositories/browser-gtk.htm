iterami/Docs.htm/Repositories/Browser.gtk
  • Installation:
    1. "git clone --depth 1 https://github.com/iterami/Browser.gtk.git"
    2. "sudo aptdcon -i libgtk-3-dev"
    3. "git clone --depth 1 https://github.com/iterami/common.git" into the same directory as "Browser.gtk".
    4. Use your terminal to navigate to the "Browser.gtk" directory and "make" it.
    5. Run the compiled "~/.iterami/browser".
  • Updating:
    1. Close "Browser.gtk".
    2. "git pull" the "common" and "Browser.gtk" repositories that you cloned.
    3. Use your terminal to navigate to the "Browser.gtk" directory and "make" it.
    4. Run the compiled "~/.iterami/browser".
  • Uninstallation:
    1. Use your terminal to navigate to the "Browser.gtk" directory.
    2. "make clean"
    3. Delete the cloned "Browser.gtk" directory.
    4. Optionally remove "common" and optionally "sudo aptdcon -r libgtk-3-dev".
    5. If you have no other iterami GTK repositories installed, you can delete the "~/.iterami/" directory.
