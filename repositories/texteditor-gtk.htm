iterami/Docs.htm/Repositories/TextEditor.gtk
  • Installation:
    1. "git clone https://github.com/iterami/TextEditor.gtk.git --depth 1"
    2. "sudo aptdcon -i libgtk-3-dev"
    3. "git clone https://github.com/iterami/common.git --depth 1" into the same folder as "TextEditor.gtk".
    4. Use your terminal to navigate to the "TextEditor.gtk" folder and "make" it.
    5. Run the compiled "~/.iterami/texteditor".
  • Updating:
    1. Close "TextEditor.gtk".
    2. "git pull" the "common" and "TextEditor.gtk" repositories that you cloned.
    3. Use your terminal to navigate to the "TextEditor.gtk" folder and "make" it.
    4. Run the compiled "~/.iterami/texteditor".
  • Uninstallation:
    1. Use your terminal to navigate to the "TextEditor.gtk" folder.
    2. "make clean"
    3. Delete the cloned "TextEditor.gtk" folder.
    4. Optionally remove "common" and optionally "sudo aptdcon -r libgtk-3-dev".
    5. If you have no other iterami GTK repositories installed, you can delete the "~/.iterami/" folder.
