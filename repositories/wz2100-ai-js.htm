  • Cheats are not used, including limiting knowledge to sensor range.
  • AIs:
    • iterami-competitive: an attempt to create a general-purpose bot that can win competitive games.
    • iterami-cyborgswarm: a bot that swarms enemies with randomized cyborgs.
    • iterami-missilefortress: a bot that researches and builds missile fortresses as quickly as possible.
  • Installation:
    1. "git clone https://github.com/iterami/WZ2100-AI.js.git --depth 1"
    2. Copy the directories of the bots you wish to use into your Warzone 2100 directory while matching the directory structure.
  • Observation:
    1. "shift + backspace" to enable debug mode and "ctrl + o" to open the debug menu.
    2. There are multiple ways to observe an AI:
      • Attach an AI script to the player you are currently observing.
      • Change the "Selected Player" to the AI you wish to observe.
    3. Click "Reveal all" to reveal the entire map for easier observation.
    4. "ctrl + +" to increase the game speed for quicker observation.
