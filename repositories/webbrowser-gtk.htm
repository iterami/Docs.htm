iterami/Docs.htm/Repositories/WebBrowser.gtk
  • Installation:
    1. "git clone https://github.com/iterami/WebBrowser.gtk.git --depth 1"
    2. "sudo aptdcon -i libgtk-3-dev webkit2gtk-4.0-dev"
    3. "git clone https://github.com/iterami/common.git --depth 1" into the same folder as "WebBrowser.gtk".
    4. Use your terminal to navigate to the "WebBrowser.gtk" folder and "make" it.
    5. Run the compiled "~/.iterami/webbrowser".
  • Updating:
    1. Close "WebBrowser.gtk".
    2. "git pull" the "common" and "WebBrowser.gtk" repositories that you cloned.
    3. Use your terminal to navigate to the "WebBrowser.gtk" folder and "make" it.
    4. Run the compiled "~/.iterami/webbrowser".
  • Uninstallation:
    1. Use your terminal to navigate to the "WebBrowser.gtk" folder.
    2. "make clean"
    3. Delete the cloned "WebBrowser.gtk" folder.
    4. Optionally remove "common" and optionally "sudo aptdcon -r libgtk-3-dev webkit2gtk-4.0-dev".
    5. If you have no other iterami GTK repositories installed, you can delete the "~/.iterami/" folder.
