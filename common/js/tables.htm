tables_add(table)
  • Adds sorting buttons to each <td> in the first child of the first child of the table, which is typically a header <tr> in a <thead> or <tbody>.
  • Ignores tables and columns with "tables-nosort" class.
  • Adds the "tables-added" class to prevent adding multiple sorting buttons to 1 table.
  • Adds a "tables-main-X" class to indicate which column contains a unique value for each row.
Arg Default Type
table HTML element
tables_format_number(value)
  • Removes commas and converts to Number during sort of "tables-numbers" columns.
Arg Default Type
value number or string
tables_init()
  • Automatically called when the file is loaded.
  • Loops thru all table elements and adds them via tables_add().
tables_sort(element, column, direction)
  • Called by sorting buttons to sort a column of a specific table.
Arg Default Type
element HTML element
column string ID
direction 1 for ascending, 0 for descending
type 1 for numbers, 0 for strings
