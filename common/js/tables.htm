tables_add(table)
  • Adds sorting buttons to the first <tr> in the <tbody>.
  • Ignores tables and columns with "tables-nosort" class.
  • Uses column with "tables-main" class as unique main column, or defaults to the first column on the left.
Arg Default Type
table HTML element
tables_format_number(value)
  • Removes commas and converts to Number during sort of "tables-numbers" columns.
Arg Default Type
value number or string
tables_init()
  • Automatically called when the file is loaded.
  • Loops through all table elements and adds them via tables_add().
tables_sort(element, column, direction)
  • Called when sorting buttons are pressed.
Arg Default Type
element HTML element
column string ID
direction 1 for ascending, 0 for descending
