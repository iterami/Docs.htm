iterami/Docs.htm/common HTM Projects/webgl.js/webgl_primitive_stars(args)
  • Randomly creates points on a sphere.
  • Additional default args are handled by webgl_prefab_args(args).
  • Translation is the center of the sphere.
  • If the "color" arg is false then a random color will be used for each point.
  • The "height-limit" arg is used to determine allowed y-values, with 1 being all y-values and 0 being none.
Arg Default Type
color [1, 1, 1, 1] array of numbers or boolean false
height-limit 1 number
radius 250 number
range 100 number
stars 100 number
