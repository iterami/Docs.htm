iterami/Docs.htm/common HTM Projects/webgl.js/webgl_scale(args)
  • Scales an entity to a specific x, y, and z value by modifying its vertices.
  • 1 is the default scale and axes with a value of false are not scaled.
  • Optionally calls webgl_entity_todo(entity) to update existing entities.
Arg Default Type
entity string ID
todo true boolean
x false boolean false or number
y false boolean false or number
z false boolean false or number
