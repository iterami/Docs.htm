iterami/Docs.htm/common HTM Projects/webgl.js/webgl_primitive_particle(args)
  • Creates a particle type and optional entities with that particle type.
  • Additional default args are handled by webgl_prefab_args(args).
  • Translation is the center of the cuboid in which the particles are created.
Arg Default Type
draw-mode 'POINTS' string
entities [] array of objects
id string ID
speed-x 0 number
speed-y 0 number
speed-z 0 number
x-max 100 number
x-min -100 number
y-max 100 number
y-min -100 number
z-max 100 number
z-min -100 number
