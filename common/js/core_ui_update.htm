iterami/Docs.htm/common HTM Projects/core.js/core_ui_update(args)
  • Updates HTML element values by ID and depending on type, but only if the value is new.
  • Can optionally update all elements that have the ID in their classes.
  • Checks for elements stored in "core_elements" before using "document.getElementById()".
Arg Default Type
class false boolean
ids {} object
