iterami/Docs.htm/common HTM Projects/chess.js/chess_threat(args)
  • Returns true if a square is threatened by the other player, or else returns false.
  • Can optionally accept a specific board as an array of strings, or else will just use the board from the game id.
Arg Default Type
board void 0 array of strings
id string
piece-x number 0-7
piece-y number 0-7
player 0 or 1
