  • Modifies or sets a character/entity property.
  • Creates the stat if it does not exist and the "has" arg is false.
  • Character "health" property has extra checks to remove attached entities upon death.
  • Modifying "vertex-colors" randomizes colors.
Arg Default Type
has true boolean
set false boolean
stat string
target character/entity object
value 1 number
