repo_drawlogic()
  • Optional function to run additional code when canvas_draw() is called.
  • No args.
repo_escape()
  • Optional function to run additional code when core_escape() is called.
  • No args.
repo_init()
  • Required.
  • No args.
  • Must call core_repo_init(args).
  • Can include additional init code for your project.
repo_logic()
  • Optional function to run additional code when utilizing canvas.js or webgl.js.
  • No args.
repo_resizelogic()
  • Optional function to run additional code when canvas_resize() is called.
  • No args.
