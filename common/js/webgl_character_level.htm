iterami/Docs.htm/common HTM Projects/webgl.js/webgl_character_level(id)
  • Attempt to return the specified character's level:
    • -3: character does not exist.
    • -2: default character level, used for characters that don't need controls and movement without paths.
    • -1: used by spectators and cameras that have access to special flying controls.
    • 0+: normal characters with controls and their own movement.
Arg Default Type
id webgl_character_id string ID
