iterami/Docs.htm/common HTM Projects/webgl.js/webgl_primitive_frustum(args)
  • Creates a frustum primitive with specified properties.
  • Translation is the center of the bottom of the frustum.
  • If the "color-bottom" or "color-top" arg arrays are empty then a random color will be used.
Arg Default Type
bottom true boolean
character webgl_character_id string ID
color-bottom [] array of numbers
color-top [] array of numbers
groups [] array of string IDs
length 2 number
middle true boolean
points 8 number
prefix entity_id_count number or string
size-bottom 2 number
size-top 2 number
top true boolean
translate-x 0 number
translate-y 0 number
translate-z 0 number
