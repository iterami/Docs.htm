iterami/Docs.htm/common HTM Projects/webgl.js Prefabs/prefabs_webgl_frustum_tree(args)
  • Creates a frustum tree prefab with the specified properties.
  • Additional default args are handled by webgl_prefab_args(args).
  • Translation is the center of the bottom face of the trunk.
Arg Default Type
bottom false boolean
height 20 number
height-range 0 number
leaf-color-bottom [.05, .15, .05, 1,] array of numbers
leaf-color-top [.1, .3, .1, 1,] array of numbers
leaf-count 3 number
leaf-points 3 number
leaf-separate 4 number
leaf-size 4 number
trunk-color [.4, .2, 0, 1,] array of numbers
trunk-points 4 number
trunk-size 1 number
