iterami/Docs.htm/common C Projects/sort.c/int sort_compare_strings_asc(const void* a, const void* b)
  • Compares two strings.
  • Used in "qsort()" rather than directly called.
  • Returns a value:
    • 0 means a is identical to b
    • <0 means a is earlier than b
    • >0 means a is later than b
Arg Type Notes
a void*
b void*
