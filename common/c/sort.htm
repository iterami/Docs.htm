int sort_compare_ints_asc(const void* a, const void* b)
  • Compares two integers.
  • Used in "qsort()" rather than directly called.
  • Returns a value:
    • 0 = integers are equal
    • less than 0 = a is less than b
    • greater than 0 = a is greater than b
